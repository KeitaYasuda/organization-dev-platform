# 組織改善プラットフォーム

組織内で改善アイデアを共有し、追跡するためのプラットフォーム。

## 開発状況
- TypeScript プロジェクトを初期構築。
- 改善項目、意見、賛同数を扱うドメインモデルを実装。
- `npm test` と `npm run build` によるビルドと型チェックを設定。
- 基本的なドメインロジックのテストを追加し、意見追加時の検証を強化。
- 依存関係をインストールし、テスト環境を整備。

## 要件概要
1. **基本機能**
   - 現在の改善項目を全社員が閲覧可能。
   - 既存項目への意見投稿および新規改善項目の提案が可能。
   - 意見に対して匿名で賛同でき、賛同数を表示。
2. **管理者機能**
   - 届いた意見を集約し、確認用にハイライト。
   - フィードバックをもとに改善項目一覧を更新。
   - どの意見がどの改善項目の参考になったかの参照関係を表示。

これらの機能は現在開発中です。
